## Project Summary

ex10x is a personal brand website for Chris Fossenier focused on teaching
AI productivity techniques, specifically "vibe coding" and AI automation.
The site offers blog content, video resources, and session booking.

--------------------------------------------------------------------------------
## Current State of the Project
--------------------------------------------------------------------------------

### Technology Stack
- Framework: Astro 5.16.3
- Styling: Tailwind CSS 4.1.17 with Typography plugin
- Build: Vite (via Astro)
- Language: TypeScript

## Decisions Made
--------------------------------------------------------------------------------

1. **Astro over other frameworks** - Chose Astro for its excellent static site
   generation, content collections for blog posts, and minimal JavaScript
   by default

2. **Tailwind CSS 4** - Using the latest Tailwind with Vite plugin for modern
   styling workflow

3. **Gray/minimal color scheme** - Clean, professional look that puts content
   first

4. **Content Collections** - Using Astro's built-in content collections for
   type-safe blog posts

## Architecture Notes
--------------------------------------------------------------------------------

### Astro Content Collections
Blog posts are in `src/content/blog/` as markdown files. The content config
at `src/content.config.ts` defines the schema. Dynamic routing via
`[...slug].astro` renders individual posts.

### Styling Approach
Using Tailwind utility classes directly in components. Global styles in
`src/styles/global.css`. Typography plugin handles blog post formatting.
