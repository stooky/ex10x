## Decisions Made
--------------------------------------------------------------------------------

1. **Batch file over bash script** - User is on Windows, so .bat is more
   native and doesn't require Git Bash or WSL

2. **Port-based process killing** - Kill by port 4321 rather than process name
   to ensure we catch any node process holding the Astro dev port

3. **taskkill /F** - Force kill to ensure clean termination
